1. **Clone the repository:**

    ```bash
    git clone https://github.com/ChandanPolai/UCHIHA-TUBE.git
    ```

2. **Install dependencies For Backend:**

    ```bash
    cd -uchiha-backend
    npm install
    ```

2. **Install dependencies For Frontend:**

    ```bash

    ```

